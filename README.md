# neuro-ai-roadmap

A working portfolio and study log at the intersection of **neuroscience, brain-computer interfaces (BCI), and AI**. This repo is not a finished product — it's evidence of work in progress, kept public so the progress is traceable.

## Why this repo exists

I'm building toward graduate study in computational neuroscience / neuroengineering / AI. My undergraduate GPA is a weakness, so this repo exists to do what a transcript can't: show concrete, dated, inspectable proof that I can read the literature, write code, and think about hard problems.

If you're reading this from a lab or admissions context — what you see here is meant to be honest, not polished. Commits over claims.
